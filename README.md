# Stack Complexity Reducer

[![Sponsor](https://img.shields.io/badge/Sponsor-GitHub%20Sponsors-ea4aaa?logo=githubsponsors)](https://github.com/sponsors/sdotwinter)

CLI tool that scans project manifests, scores stack complexity, and generates a simplification plan.

## Usage
```bash
python3 main.py scan examples
python3 main.py score examples
python3 main.py simplify-plan examples
python3 main.py simplify-plan examples --json
```
